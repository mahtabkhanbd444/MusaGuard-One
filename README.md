# Musa Guard DNS

Musa Guard DNS is a free, family & Islamic safe DNS blocklist project.

## Blocks
- Ads & Trackers
- Adult / Porn websites
- Gambling & Betting
- Haram content
- Malware & Phishing

## Designed for
- Android Private DNS
- AdGuard Home
- NextDNS
- Pi-hole

## Maintained by
Community & open-source blocklists
